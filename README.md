# 像素办公室 - Pixel Office

一个像素风格的放置观赏类网页游戏，将 Claude Code 的编程进程可视化为办公室里的员工。

## 功能特性

- 🎮 **像素艺术风格** - 程序化生成的精美像素角色和办公室环境
- 👥 **5种不同角色** - 每种角色有独特的颜色和外观
- 🔄 **状态动画系统** - 工作、摸鱼、下班三种状态动画
- 📊 **实时进程状态** - 3秒轮询更新进程状态
- 🖱️ **交互式提示** - 鼠标悬停显示进程信息

## 快速开始

```bash
npm install
npm run dev
# 打开 http://localhost:5173
```

## 键盘控制

- **A** - 添加新进程
- **R** - 移除最后一个进程

## 构建部署

```bash
npm run build
npm run preview
```

## 技术栈

- **Phaser 3** - 游戏引擎
- **TypeScript** - 类型安全
- **Vite** - 快速构建工具
